# 30-Second Deployment Proof Video Script

## Setup
- **Tool:** QuickTime (Mac), Game Bar (Windows), or OBS Studio
- **Resolution:** 1080p minimum
- **Audio:** Optional (can be silent with text overlays)
- **Length:** 30-45 seconds max

## Script

**[0:00 - 0:05]** 
- Open browser
- Show Google Cloud Console login
- **Text Overlay:** "Google Cloud Console"

**[0:05 - 0:12]**
- Navigate to Cloud Run
- Show services list
- Click on "gemini-scraper-agent"
- **Text Overlay:** "Cloud Run Service"

**[0:12 - 0:20]**
- Show service overview with green "SERVING" status
- Point cursor to service URL
- Highlight metrics showing activity
- **Text Overlay:** "✅ Live & Serving"

**[0:20 - 0:28]**

- Scroll through live logs
- Show timestamps and API calls
- **Text Overlay:** "Real-time logs"

**[0:28 - 0:35]**
- Click service URL
- Show app loading and running
- **Text Overlay:** "Live Application"

**[0:35 - 0:40]**
- Quick show of Vertex AI dashboard (optional)
- **Text Overlay:** "Powered by Gemini 2.0"

**[0:40 - 0:45]**
- End screen with:
  - Project name
  - GitHub URL
  - #GeminiLiveAgentChallenge
