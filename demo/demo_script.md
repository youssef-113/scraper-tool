## 🎥 RECORDING TIPS

### Equipment
- Use OBS Studio or ScreenFlow
- Record at 1080p, 30fps
- Use good microphone (Blue Yeti or similar)
- Quiet environment

### Voice
- Speak clearly and enthusiastically
- Pause for 1 second between sections
- Emphasize key features
- Natural tone, not robotic

### Visuals
- Clean desktop, close unnecessary apps
- Use cursor highlighting
- Add text overlays for key points
- Smooth transitions (fade or cut)

### Post-Production
- Add background music (low volume)
- Color grade for consistency
- Add captions for accessibility
- Export as MP4, H.264, 1080p
