## Emergency Checklist (If something breaks last minute)

### Service Down
1. Check Cloud Run logs
2. Redeploy: `./gcp/deploy.sh` 
3. Update service URL if changed
4. Re-test everything

### Video Upload Fails
1. Try YouTube upload
2. Try Vimeo as backup
3. Upload to GitHub repo as .mp4
4. Use GitHub raw URL in Devpost

### GitHub Issues
1. Check all files pushed
2. Verify repo is public
3. Test clone in incognito
4. Push any missing files
